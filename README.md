# Personal Finance Tracker

A full-stack personal finance management application that helps users track income, expenses, savings, and budgets through interactive analytics and visualizations.

## Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Secure User-Specific Data

### Transaction Management
- Add Transactions
- Edit Transactions
- Delete Transactions
- Income & Expense Support
- Multiple Categories

### Analytics
- Current Balance Tracking
- Income Analytics
- Expense Analytics
- Savings Calculation
- Top Spending Category
- Largest Expense Tracking
- Recent Transaction Tracking

### Monthly Insights
- Monthly Income
- Monthly Expenses
- Monthly Savings
- Savings Rate

### Budget Tracking
- Set Monthly Budget
- Budget Usage Percentage
- Remaining Budget Calculation
- Budget Status Indicators

### Charts & Visualization
- Radial (Doughnut) Expense Chart
- Monthly Income vs Expense Bar Chart
- 6-Month Financial Trend Line Chart

### Filtering
- Category Filter
- Income / Expense Type Filter

### Productivity Features
- Undo Transactions
- Redo Transactions
- Live Date & Time Display

## Tech Stack

### Frontend
- HTML
- CSS
- JavaScript
- Chart.js

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Tokens)
